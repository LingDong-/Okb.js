vector

Utility for manipulating vectors in 2D/3D space. Both Array-style and Object-with-xyz-fields style vectors are accepted, e.g.: {x:1, y:2}, {x:1, y:2, z:3}, [1, 2, 3], [1, 2]

vector
Properties
forward (Object) : unit vector in positive z direction (0,0,1)
up (Object) : unit vector in positive y direction (0,1,0)
right (Object) : unit vector in positive x direction (1,0,0)
Static Members
▸ vector(x, y, z, like)

Create a new vector

vector(x: number, y: number, z: number, like: (Object | Array<number>)): (Object | Array<number>)
Parameters
x (number) the x coordinate
y (number) the y coordinate
z (number) the z coordinate, put undefined or omit the argument to create a 2D vector
like ((Object | Array<number>)) example vector, if specified, the generated vector will have the same dimension and data structure
Returns
(Object | Array<number>): [x,y(,z)] | {x:x,y:y(,z:z)}
Example
var v = vector.vector(1,2,42);        // {x:1, y:2, z:42}
var v = vector.vector(1,2);           // {x:1, y:2}
var v = vector.vector(1,2,42,[0,0,0]) // [1, 2, 42]
var v = vector.vector(1,2,42,[0,0])   // [1, 2]
▸ rotateAxial(vec, axis, th)

Rotate a vector around axis by an angle

rotateAxial(vec: (Object | Array<number>), axis: (Object | Array<number>), th: number): (Object | Array<number>)
Parameters
vec ((Object | Array<number>)) the vector to rotate
axis ((Object | Array<number>)) a vector representing axis of rotation
th (number) the rotation angle in radians
Returns
(Object | Array<number>): a new vector
Example
var v = vector.vector(1,2,3);                            // {x:1, y:2, z:3}
var u = vector.rotateAxial(v, vector.up, Math.PI/2);     // {x:3, y:2, z:-1}
var w = vector.rotateAxial([1,2,3], [0,1,0], Math.PI/2); // [3, 2, -1]
